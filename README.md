# Лабораторна робота №7: React. Компоненти, стан, властивості

## Опис проєкту

Цей проєкт є React-застосунком, що складається з двох частин:
1.  **Реалізація функціоналу Лабораторної роботи №4** з використанням компонентного підходу React. Включає функціональні та класові компоненти, керування станом та передачу функцій через props.
2.  **Створення галереї товарів** з використанням багаторазового компонента `GoodsCard` та передачі даних через props.

### Використані концепції React

*   **Компоненти:** Застосунок розділено на логічні частини (`Header`, `Content`, `Image`, `GoodsCard`).
*   **Функціональні та Класові компоненти:** `Header`, `Image`, `GoodsCard` реалізовані як функціональні компоненти, а `Content` — як класовий.
*   **Стан (State):** Використовується хук `useState` в `App.js` для керування зображенням та `this.state` в `Content.js` для підсвічування елементів.
*   **Властивості (Props):** Дані та функції-обробники передаються від батьківських компонентів до дочірніх.
*   **Рендеринг списків:** Галерея товарів генерується динамічно за допомогою методу `.map()`.
*   **JSX:** Розмітка створюється за допомогою синтаксису JSX.

## Локальний запуск

Щоб запустити проєкт локально, виконайте наступні кроки:

1.  **Клонуйте репозиторій:**
    ```bash
    git clone https://github.com/Ratarekko/Front-End-Lab7
    ```

2.  **Перейдіть до папки проєкту:**
    ```bash
    cd Front-End-Lab7
    ```

3.  **Встановіть залежності:**
    ```bash
    npm install
    ```

4.  **Запустіть застосунок:**
    ```bash
    npm start
    ```

Після цього проєкт відкриється у вашому браузері за адресою `http://localhost:3000`.
